# Cliente OP

Frontend React para el proyecto final de encuestas de la Oficina de Planeamiento.

## Primer corte

- Ruta `/login` con validación local de correo y contraseña.
- Ruta protegida `/app` con sesión mock en memoria.
- Shell inicial preparado para los próximos módulos: encuestas, pendientes, mapa y usuarios.
- Sin integración con backend, WebSocket, REST, gRPC, cámara ni almacenamiento offline.

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run build
```
